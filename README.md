# codex-skills

Reusable custom Codex skills.

## Skills

- `codex-session-memory`: local persistent memory workflow for long-running work (`search -> timeline -> get`).

## Add New Skill

Create a folder under repo root with this minimum shape:

```
<skill-name>/
  SKILL.md
```

Optional:

- `scripts/`
- `references/`
- `assets/`
- `agents/openai.yaml`
